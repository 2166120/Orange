NOTE: If you are going to write JavaScript (JS) as an external source, always write the following: 

<script type = "text/javascript" src = "JS-program-name.js"></script>

If JS is going to manipulate a part of Hypertext Markup Language (HTML), then put the following within the body of an HTML file: 

<div id="id-name"></div>

The form of the main method in JS that will be used for the execution of the program: 

function init()
{
  var panel = document.getElementById("id-name");
  //insert executable code here
}
document.addEventListener("DOMContentLoaded", init, false);


Employing Objects

Object is an entity with properties (variables or constants) that store data and methods (functions) that perform tasks.

Types of Objects

  • Built-in Objects
    - objects with useful features that are part of the JS language
  • Document Object Model (DOM) Objects
    - objects that are web page components with hierarchical arrangement
  • Custom Objects
    - customized objects made by the author for storing data and providing functionalities

Creating an Object

Object Definition Syntax: 

function Object-name(parameter1, parameter2)
{
  this.first-property = parameter1;
  this.second-property = parameter2;

  this.first-method = function1;
  this.second-method = function2;
}

Now we will make a JS file "Fungus.js". The object will be named Fungus. Its properties in this program will be cell-type, cell-number and message. To get the values of cell-type, cell-number and message, we will also employ the methods get-cell-type, get-cell-number and get-message. We will set the message so there will be a set-message. The program to be written will be the following: 

function Fungus(message)
{
  this.cell-type = "Eukaryotic";
  this.cell-number = "Multi";
  this.message = message;
  this.get-cell-type = get-cell-type;
  this.get-cell-number = get-cell-number;
  this.get-message = get-message;
  this.set-message = set-message;
}

For the methods to work, we will also give them the functions that returns the values of cell-type and cell-number in the following: 

function get-cell-type() { return cell-type; }
function get-cell-number() { return cell-number; }
function get-message() { return message; }
function set-message( message ) { this.message = message; }

Extending an Object

In this tutorial, we are going to extend object Fungus with the prototype property for the purpose of adding properties and methods that will be used for a specific instance of Fungus called Mushroom. Its additional properties will be button-color, button-size, stem-color, stem-size, has-spores and spores-color. These properties will also have their own get and set methods. The changes to script "Fungus.js" will be saved to a new file named "Mushroom.js". We will also make an html file named "Mushroom.html" with a div within its body. The value of the id attribute of div will be mushroom-js. The extensions of Fungus to be written below the Fungus function will be the following: 

Fungus.prototype.button-color = [];
Fungus.prototype.button-size = [];
Fungus.prototype.stem-color = [];
Fungus.prototype.stem-size = [];
Fungus.prototype.has-spores = [];
Fungus.prototype.spores-color = [];

The Fungus extensions are arrays that will be discussed in the next subtopic of Employing Objects. We will implement an init function below the Fungus extensions this time. 

function init()
{
  var panel = document.getElementById("mushroom-js");
  var Mushroom = new Fungus("I am a cute baby Mushroom!");
}
document.addEventListener( "DOMContentLoaded", init, false );
