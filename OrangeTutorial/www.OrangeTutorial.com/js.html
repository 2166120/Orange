Objects

Object is an entity with properties (variables or constants) that store data and methods (functions) that perform tasks.

Types of Objects

  • Built-in Objects
    - objects with useful features that are part of the JS language
  • Document Object Model (DOM) Objects
    - objects that are web page components with hierarchical arrangement
  • Custom Objects
    - customized objects made by the author for storing data and providing functionalities

Creating an Object

Object Definition Syntax: 

function Object-name(parameter1, parameter2)
{
  this.first-property = parameter1;
  this.second-property = parameter2;

  this.first-method = function1;
  this.second-method = function2;
}

Now we will make a JS file "Fungus.js". The object will be named Fungus. Its properties in this program will be cell-type, cell-number and message. To get the values of cell-type, cell-number and message, we will also employ the methods get-cell-type, get-cell-number and get-message. We will set the message so there will be a set-message. The program to be written will be the following: 

function Fungus(message)
{
  this.cell-type = "Eukaryotic";
  this.cell-number = "Multi";
  this.message = message;
  this.get-cell-type = get-cell-type;
  this.get-cell-number = get-cell-number;
  this.get-message = get-message;
  this.set-message = set-message;
}

For the methods to work, we will also give them the functions that returns the values of cell-type and cell-number in the following: 

function get-cell-type() { return cell-type; }
function get-cell-number() { return cell-number; }
function get-message() { return message; }
function set-message( message ) { this.message = message; }
