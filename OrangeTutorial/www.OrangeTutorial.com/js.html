JS: JavaScript

NOTE: If you are going to write JavaScript (JS) as an external source, always write the following: 

<script type = "text/javascript" src = "JS-program-name.js"></script>

If JS is going to manipulate a part of Hypertext Markup Language (HTML), then put the following within the body of an HTML file: 

<div id="id-name"></div>

The form of the main method in JS that will be used for the execution of the program: 

function init()
{
  var panel = document.getElementById( "id-name" );
  //insert executable code here
}
document.addEventListener( "DOMContentLoaded", init, false );


Objects

Object is an entity with properties (variables or constants) that store data and methods (functions) that perform tasks.

Types of Objects

  • Built-in Objects
    - objects with useful features that are part of the JS language
  • Document Object Model (DOM) Objects
    - objects that are web page components with hierarchical arrangement
  • Custom Objects
    - customized objects made by the author for storing data and providing functionalities


Creating an Object

Object Definition Syntax: 

function Object-name(parameter1, parameter2)
{
  this.first-property = parameter1;
  this.second-property = parameter2;

  this.first-method = function1;
  this.second-method = function2;
}

The Object Definition function is also called a Constructor. Now we will make a JS file "Fungus.js". The object will be named Fungus. Its properties in this program will be cell-type, cell-number and message. To get the values of cell-type, cell-number and message, we will also employ the methods get-cell-type, get-cell-number and get-message. We will set the message so there will be a set-message. The program to be written will be the following: 

function Fungus(message)
{
  this.cell-type = "Eukaryotic";
  this.cell-number = "Multi";
  this.message = message;
  this.get-cell-type = get-cell-type;
  this.get-cell-number = get-cell-number;
  this.get-message = get-message;
  this.set-message = set-message;
}

For the methods to work, we will also give them the functions that returns the values of cell-type and cell-number in the following: 

function get-cell-type() { return cell-type; }
function get-cell-number() { return cell-number; }
function get-message() { return message; }
function set-message(message) { this.message = message; }

Extending an Object

Since not all fungi have the properties of a mushroom, we are going to extend object Fungus with its prototype property for the purpose of adding properties and methods outside of Fungi constructor that will be used for a specific instance of Fungus called Mushroom. Its additional properties that will be written below the Constructor are button-color, button-size, stem-color, stem-length, has-spores and spores-color. Their get methods are get-button-color, get-button-size, get-stem-color, get-stem-size, get-has-spores and get-spores-color. Their set methods are set-button-color, set-button-size, set-stem-color, set-stem-size, set-has-spores and set-spores-color. All types of objects have the prototype property in JavaScript. The additional properties and methods part of the program below will be written below the Fungus Constructor. 


Fungus.prototype.button-color = "red";
Fungus.prototype.button-size = "big";
Fungus.prototype.stem-color = "yellow";
Fungus.prototype.stem-length = "short";
Fungus.prototype.has-spores = true;
Fungus.prototype.spores-color = "white";

Fungus.prototype.get-button-color = function() { return button-color; };
Fungus.prototype.get-button-size = function() { return button-size; };
Fungus.prototype.get-stem-color = function() { return stem-color; };
Fungus.prototype.get-stem-length = function() { return stem-length; };
Fungus.prototype.get-has-spores = function() { return has-spores; };
Fungus.prototype.get-spores-color = function() { return spores-color; };

Fungus.prototype.set-button-color = function(button-color) {  this.button-color = button-color; };
Fungus.prototype.set-button-size = function(button-size) {  this.button-size = button-size; };
Fungus.prototype.set-stem-color = function(stem-color) {  this.stem-color = stem-color; };
Fungus.prototype.set-stem-length = function(stem-length) {  this.stem-length = stem-length; };
Fungus.prototype.set-has-spores = function(has-spores) {  this.has-spores = has-spores; };
Fungus.prototype.set-button-color = function(spores-color) {  this.spores-color = spores-color; };

There you go! Here is a cute mushroom with the properties above.
